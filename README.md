# Trainer Coins

Trainer Coins is a hold-focused crypto token inspired by classic game progression.
The project emphasizes ranking, status, and long-term commitment rather than frequent utility actions.

## Purpose
Trainer Coins is designed to be stacked and held, with community visibility provided through:
- A public ranking website
- A team generator that reflects holder status

## Utility
- Wallet-based ranking system
- Team generator tied to rank
- Community showcase tools

## Roadmap
- Rank website (MVP)
- Team generator (MVP)
- Visual rank tiers
- UI polish & deployment
